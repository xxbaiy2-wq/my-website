OfferPilot — Career Radar

AI Job-Search Decision Agent — a full-lifecycle job-hunting assistant for students and early-career professionals. Unlike traditional resume tools or interview bots, OfferPilot is a multi-agent system with autonomous decision-making capabilities — understanding user profiles, sensing market trends, diagnosing skill gaps, and autonomously planning and executing job-search strategies.

Key Features

  1. Smart Analysis — 5-step Pipeline (Profile → Market → Skills → Optimization → Decision), one-click job-search decision report
  2. AI Mock Interview — text + voice dual modes, 6 interview styles, auto-generated follow-up questions based on resume
  3. JD-Targeted Resume Optimization — paste target job description, auto-calculate match score and provide targeted revision suggestions
  4. Career Radar — search 30+ real web pages via Bing, extract structured data across 7 dimensions: salary, skills, education, etc.
  5. Resume Authenticity Audit — Truth Guard Agent auto-detects resume inflation risks and provides revision suggestions

Tech Architecture

Backend: FastAPI + Python Multi-Agent (Web Search Agent, Smart Analysis Agent, Resume Agent, etc.) + LLM Integration (Mimo v2.5)
Frontend: React SPA + Web Speech API (voice interview) + ECharts WordCloud
Deployment: Nginx reverse proxy + RESTful API

Highlights

  • Multi-agent autonomous decision-making: LLM generates search queries → 28+ queries → 30+ pages crawled → 7-dimension data extraction → LLM report generation
  • Fully async non-blocking: asyncio.to_thread() ensures SSE event stream is never blocked by LLM calls
  • LLM timeout protection: 180s SIGALRM + rule-engine fallback to ensure service availability
